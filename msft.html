Automated Financial Reporting with AI

As part of the Microsoft Business Analytics Capstone, I led the development of an end-to-end automation pipeline that transformed manual, time-consuming financial reporting processes into a fully automated solution. The focus was on streamlining month-end financial analysis for accounts receivable, including both data visualization and commentary generation.

Project Objective

  • Microsoft’s finance team was manually preparing recurring PowerPoint presentations summarizing accounts receivable performance.
  • Each cycle involved data extraction, cleansing, analysis, chart generation, and drafting narrative commentary—consuming dozens of analyst hours per month.
  • The project aimed to automate both the visual and narrative components of the reporting pipeline to save time, increase consistency, and reduce error.

Workflow

  • Data extraction: Accounts receivable data was exported from internal financial systems as structured Excel files.
  • Data preprocessing: Cleaned and structured the raw Excel data using Python’s pandas and numpy libraries.
  • Data visualization: Automated generation of summary charts using matplotlib and seaborn to visualize aging, overdue accounts, and trends.
  • Automated narrative generation: Used OpenAI language models and prompt engineering to generate written analysis based on key metrics and trends in the data.
  • Report assembly: Used the python-pptx library to automate creation of formatted PowerPoint slides combining both charts and AI-generated commentary.
  • Delivery: Delivered reusable, end-to-end automation pipeline that produced fully formatted financial reports with minimal human intervention.

Results & Business Impact

  • Reduced manual reporting effort by over 130 hours per reporting cycle, significantly improving reporting efficiency.
  • Enabled faster delivery of financial insights to leadership, improving decision-making speed during monthly reviews.
  • Standardized report formatting and commentary generation, reducing inconsistencies across reporting cycles.
  • Demonstrated successful integration of generative AI and automation tools into financial reporting workflows, laying the foundation for future automation initiatives.

Technical Stack

  • Python (matplotlib, bs4, pptx, pandas, numpy)
  • OpenAI API (for generative AI-powered financial commentary)
  • Microsoft Excel (data preparation and validation)
  • SQL (data extraction and initial data preparation)
  • PowerPoint (automated slide deck generation)

Index

This project demonstrates how blending AI-powered narrative generation, Python automation, and financial domain expertise can radically improve reporting efficiency. By automating both data visualization and executive commentary, the finance team was able to accelerate their reporting cycle while improving consistency and freeing analysts to focus on strategic insights.
